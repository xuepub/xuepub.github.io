启用 .htaccess 实现 WordPress 固定链接

在 Ubuntu Apache2 的 WordPress 网站中，固定链接需要如下设置：

  1. 启动 mod_rewrite
    • sudo a2enmod rewrite
    • sudo service apache2 restart
  2. 启用 .htaccess
    • 修改 apache2 配置文件，sudo nano /etc/apache2/sites-enabled/000-default.conf 其中内容为 <Directory /var/www/html> Options Indexes FollowSymLinks MultiViews AllowOverride All Order allow,deny allow from all
    • sudo service apache2 restart
  3. 按 WordPress 设置 - 固定链接 - 自定义链接给出的提示，添加 .htaccess 至相应根目录

Ubuntu LAMP 部署

系统

  • 阿里云，购买并开通 Ubuntu 服务器
  • Digital Ocean， 开通 Ubuntu 服务器
  • 腾讯云，购买并开通 Ubuntu 服务器
    • 由于不提供 root 密码，需要进入 ubuntu 账号后，使用sudo passwd root 更改密码
    • sudo vi /etc/ssh/sshd_config，修改 PermitRootLogin 为 yes，保存退出
    • 重启 ssh，sudo service ssh restart
  • 虚拟机 vmware，中安装 ubuntu desktop / server

  下载工具

  • XShell 用于远程访问服务器的终端
  • WinSCP 用户远程访问服务器的文件系统

  终端基本命令

  • cd 目录操作
  • ls 列出当前文件
  • service –status-all 查看系统当前进程

  安装LAMP

  1. sudo apt-get update sudo apt-get install apache2
  2. sudo apt-get install mysql-server php5-mysql sudo mysql_install_db sudo mysql_secure_installation
  3. sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt

  启动 apache mysql

  • service apache2 start/restart/stop
  • service mysqld start/restart/stop

  测试 php 环境

  1. 创建一个 .php 文件，内含代码
  2. 将该文件放入 apache 的文档根目录 /var/www/html/ 中

  安装 phpmyadmin

  1. sudo apt-get install wget unzip
  2. 在 html 根目录下执行 wget <phpmyadmin 下载地址>
  3. unzip <phpmyadmin  下载包>

  安装wordpress

  1. wget <wordpress 下载地址>
  2. unzip <wordpress  下载包>
  3. 安装

  vi 基本操作

  • ESC 进入命令行模式（退出编辑）
  • i 进入插入模式（可编辑）
  • :wq 保存文件并退出
  • :q 直接退出文件

在 Windows 中安装 wordpress

  1. 下载安装 xampp，修改 mysql 密码 shell 中键入：mysqladmin -u root -p <原密码> password <新密码> 同时修改 phpMyAdmin 配置文件中的登录密码
  2. 启动 apache、mysql
  3. 下载 wordpress 源文件至 htdocs 目录下，解压至该目录
  4. 浏览器中通过 localhost/wordpress 访问并安装
  5. 在 phpMyAdmin 中创建新数据库，使用 utf8_general_ci 排序规则
  6. 安装 wordpress
  7. 安装成功，通过后台 localhost/wordpress/wp-admin 进行管理
