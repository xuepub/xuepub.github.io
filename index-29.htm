Java 准备

Java SE – 标准版，包含全部语法、基本库

Java EE – 企业版，用于 Web、服务等

JVM – 虚拟机，用于运行编译过的 Java、Scala、Groovy、Clojure 等代码

 

 

Java EE 中包含了Java SE JDK 开发包（必备），后者内含 JRE

JDK 安装后需要配置 JAVA 环境变量

http://www.cnblogs.com/zhj5chengfeng/archive/2013/01/01/2841253.html

集成开发环境（IDE）

  • Eclipse 系列（包含 Spring Tool Suite）
  • IntelliJ IDEA 系列（包含 Android Studio）
