Bootstrap 基础

推荐阅读

Bootstrap 官方中文文档 http://v3.bootcss.com/ W3school 相关文档 http://w3schools.bootcss.com/bootstrap/ 《Bootstrap实战》 中文版  

有效文件

bootstrap/ ├── css/ │ └── bootstrap.min.css ├── js/ │ └─── bootstrap.min.js ├── fonts/  

引入

在 中导入 bootstrap.min.css，注意在自定义 CSS 文件之前； 在 结束部分引入  bootstrap.min.js，注意在引入 jquery.min.js 文件之后； 可以考虑使用 CDN 公共库。  

提供功能

  • 各浏览器 reset
  • 移动优先 responsive design
  • 栅格系统
  • 导航栏
  • 表格和表单样式
  • 模态框
  • 标签页
  • 图片轮播
  • 响应式图片

 

相关组件

  • jQuery（必须使用）
  • jQuery UI
  • Font Awesome 图标

 

使用要点

移动优先 必须使用 Bootstrap 要求的三行 以支持各种设备。   ** js 文件后置引入** 据称在  的结束部分引入 js 文件能够优化网页加载速度，提升动画效果，业界普遍使用。 注意这种情况下 DOM 事件的写法，比如无法直接在 HTML 的 button 标签中使用 onclick 事件。   顶部导航 nav 格式需按 Bootstrap 文档来。 移动端导航通过下拉式菜单弹出，其位置在 div.navbar-header>button.navbar-toggle.collapsed，注意 button 中 data-target 属性值为 #main-nav，main-nav 为正常界面下主菜单的 id 值，要保证 Bootstrap 的 js 文件和其依赖的 jQuery 文件正常引用。 二级菜单的写法类似移动端下拉式菜单。   响应式  responsive 1. container 所有网站上的内容，都放到

中，这样页面宽度自动支持响应式设计：

  • lg 大屏幕 x >1200px，1170px container 宽度
  • md 中屏幕 992px < x < 1200px，970px container 宽度
  • sm 小屏幕 768px < x < 992px，750px container 宽度
  • xs 手机屏幕  x < 768px ，auto container 宽度
