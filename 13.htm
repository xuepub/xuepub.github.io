文档

https://jhipster.github.io/

https://www.infoq.com/minibooks/jhipster-mini-book

http://www.jhipster-book.com/

 

环境安装

  • git
  • Node.js LTS 4.4
  • npm install -g npm（MacOS 前置 sudo）
  • npm install -g yo
  • npm install -g bower
  • npm install -g gulp
  • maven 会自动下载
    • gradle 可选

 

Yeoman

  • Yo 脚手架工具，可从模板生成标准的项目文件
  • Bower 包管理工具，管理并下载前端依赖
  • Gulp 构建工具，实现前端的多种任务

发布

开发

运行启动 8080 端口

  • mvnw （mac ./mvnw）
  • 或 gradlew bootrun

gulp 启动9000端口，监控前端代码变化

  • gulp serve

 

产品

运行

  • mvnw -Pprod
  • 或 gradlew -Pprod

打包并部署

  • mvnw -Pprod package
  • 或 gradlew -Pprod bootRepackage
  • java -jar projectname-0.0.1-SNAPSHOT.war –spring.profiles.active=prod
