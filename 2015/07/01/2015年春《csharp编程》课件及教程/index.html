2015春《C#编程》课件及教程

课件及教程下载

http://pan.baidu.com/s/1bClypS  

课件说明

来自 telerikacademy.com，是2013年的 C# 公开课程，包括 C# 基础和 WPF 两部分。 保加利亚 Telerik 公司研发了著名的第三方界面库 Telerik 系列产品。 课件的中文翻译，已在2015年春的 C# 教学活动（72课时）中使用。 Telerik JavaScript 教程的中文翻译，见 http://www.xuepub.com/74.html

Karel教程说明

《Karel C#编程入门》是娱乐之作，供学习参考，习题参考了《Karel J Robot》。 另一位作者，具有20年软硬件开发经验的软件架构师，参与过 .NET Framework 标准的制定工作，并曾担任某第三方界面库的首席架构师。 同时，作者开发了 Karel 的 js 版本，包括采用 Node.js 为后端的出题测试系统： Source Code: https://github.com/lambdax-xx/karel.js （可自行部署） Demo: http://kareljs.org/ （未完全启用）    

推荐阅读

《C# 基础视频》 刘铁锰录制，https://www.youtube.com/user/waspec/videos （需要翻墙） 《C# x.0 Pocket Reference》 O’reilly 出版社，英文 C# 语法口袋书，x ∈ {4, 5, 6} 《深入理解 C#（第三版）》 Manning 出版社，讲解 C# 高级语法特性 《敏捷软件开发：原则、模式与实践（C#版）》，马丁福勒，大师出品
