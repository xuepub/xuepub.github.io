Python 应用

on 2016年4月14日
  • 应用
by szzz

本文翻译自 Python 官网

Python Package Index 包索引列表，有大量的第三方模块

Web 和互联网开发

Web 开发：

  • 框架 Django Pyramid
  • 微框架 Flask Bottle
  • 内容管理系统 Plone django CMS

Python 标准库支持大量协议

  • HTML and XML
  • JSON
  • E-mail processing
  • FTP IMAP 以及其他Internet protocols
  • socket interface

以及：

  • Requests 强大的 HTTP 客户端库
  • BeautifulSoup HTML 解析器可处理所有古怪的 HTML
  • Feedparser 用于解析 RSS/Atom 源
  • Paramiko 实现 SSH2 协议
  • Twisted Python 异步网络编程框架

 

科学和数值计算

Python 广泛的用于 scientific and numeric 计算：

  • SciPy 是数学科学和工程集成包
  • Pandas 是数据分析和建模库
  • IPython 强大的交互式环境，易于编辑并记录工作过程，支持虚拟化和并行计算
  • Software Carpentry Course 上有一些科学计算教程

 

教育

Python 是用于教学的超级语言，跨越了入门级和专业级

  • 参考书  How to Think Like a Computer Scientist, Python Programming: An Introduction to Computer Science, Practical Programming.
  • Education Special Interest Group 是讨论教学的好去处

 

桌面 GUIs

Tk GUI 库包含了 Python 最多的二进制发行包

一些工具在许多平台上可用：

  • wxWidgets
  • Kivy 用于写多点触控
  • Qt 通过 pyqt 或 pyside

特定平台工具：

  • GTK+
  • win32 extensions

 

软件开发

Python 常用于软件开发者的支持语言，用于构建控制和管理

  • SCons 用于构建控制
  • Buildbot 和 Apache Gump 用于自动化持续编译和测试
  • Roundup 或 Trac 用于 bug 追踪管理
