Ionic 简述

  Ionic 用于开发基于 HTML5 的混合移动应用  

参考

开源站点 http://ionicframework.com/ 付费版 http://www.ionic.io/  

准备

首先安装 NodeJS https://nodejs.org/

$ npm install -g cordova ionic

Cordova http://cordova.apache.org/ 用于本地构建移动应用  

本地构建

Android 下载安卓 SDK http://www.android-studio.org/ 无需安装 Android Studio 配置 SDK https://cordova.apache.org/docs/en/latest/guide/platforms/android/index.html 可使用模拟器，或打包为 apk   iOS 使用 macOS，Xcode  

开发

界面使用了 AngularJS http://www.xuepub.com/490.html 界面生成器 http://ionic.io/products/creator/ 可用于生成原型 最佳实践 https://github.com/mwaylabs/generator-m-ionic
