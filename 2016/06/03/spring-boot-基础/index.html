基本代码分析

com
+- example
+- myproject
+- Application.java - 1
+- domain
+- Customer.java - 2
+- CustomerRepository.java - 3
+- service
+- CustomerService.java - 4
+- web
+- CustomerController.java - 5

  **Application.java - 1 ** 入口 Main 方法，执行配置及服务 类注解 @SpringBootApplication 等同于 @Configuration 配置类 + @EnableAutoConfiguration + @ComponentScan 自动扫描 @Configuration 使用基于 XML 的配置 @ComponentScan 用于自动查找所有以 @Component、@Service、@Repository、@Controller 注册的 Bean 方法注解 @Autowired 指定构造方法注入的依赖   domain/Customer.java - 2 POJO，即领域模型 类注解 @Entity 存储数据对象，定义属性、 get / set 方法，可通过构造方法 set 属性   domain/CustomerRepository.java - 3 Spring Data JPA Repositories 数据访问 继承自 Repository<Customer, Long> 或 CrudRepository<Customer, Long> 接口   service/CustomerService.java - 4 服务类 / 业务逻辑类 - 封装功能 类注解 @Service   web/CustomerController.java - 5 Spring Web MVC 控制器 类注解 @Controller 或 @RestController 类 / 方法注解 @RequestMapping 指定路由 返回视图（JSP、模板）或 Json（前后分离）  

配置

CustomerConfig.java 支持基于 Java 和 XML 的配置 类注解 @Configuration 声明配置类，用于定义 Bean 类注解 @ComponentScan 自动扫描 方法注解 @Bean 声明配置类中方法的返回值为 Bean   application.properties 功能参数配置  

数据

http://blog.netgloo.com/2014/10/27/using-mysql-in-spring-boot-via-spring-data-jpa-and-hibernate/ Spring Data JPA 参考 http://docs.spring.io/spring-data/jpa/docs/1.10.1.RELEASE/reference/html/ 参数自动配置 / Java 手动配置 DataSource 声明实体类，@Entity 声明对应的 Dao 接口，继承 CrudRepository 接口 在控制器中指定 Dao，并操纵数据对象  

认证
