2015秋《前端开发》课件

课件下载

http://pan.baidu.com/s/1slVANmd  

课件说明

翻译自 Telerik 2014 年前端开发多项课程，Telerik 开发了著名的商业界面库 Kendo UI HTML Framework， 本课件几乎全面介绍了 HTML CSS 以及 JS 的全部知识点，并在讲解 JS 基础时考虑到了学生的 C# 背景。

教学说明

除了使用 Telerik 的课件（60%）之外，在 2015 秋的《前端开发》课程教学活动中还使用了其他资料： HTML CSS：

  • W3school 中文站的 HTML 和 CSS 部分
  • 《CSS禅意花园》和禅意花园网站 http://www.csszengarden.com/，CSS 显示技术
  • Bootstrap，前端 UI 框架

JavaScript：

  • Firefox MDN https://developer.mozilla.org/zh-CN/docs/Web/JavaScript，JavaScript 基本概念及 ES2015+ 讲解

  • 《JavaScript语言精粹（修订版）》Douglas Crockford 著，规范的 JavaScript 语法

  • JavaScript 秘密花园 http://bonsaiden.github.io/JavaScript-Garden/zh/

  •  http://todomvc.com/ 前端 MVC

  • Curran Kelleher  在 Youtube 上的视频和 Github 上的幻灯 https://github.com/curran/screencasts Backbone、AngularJS、D3.js

  • PhoneGap 移动端的前端技术
