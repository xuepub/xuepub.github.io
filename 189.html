MySQL 基础

on 2016/04/25
  • 应用
by szzz

术语

  • Database（数据库）数据集总容器
  • Table（表）实际存储数据
  • Row（行）表中的一条记录，包含几个字段
  • Colome（列）表中字段名

 

访问方式

  • 命令行
  • PhpMyAdmin（Web 可视管理工具）
  • 使用 PHP Python 等编程语言

 

常用命令

数据命令
  • ALTER
  • CREATE
  • DELETE
  • DROP
  • INSERT
  • SELECT
  • TRUNCATE
  • UPDATE
  • USE
管理命令
  •  BACKUP
  • \c
  • DESCRIBE
  • EXIT QUIT
  • GRANT
  • RENAME
  • SHOW
  • LOCK UNLOCK

 

一个简单实例

容器（数据库和表）

创建并使用一个书架 bookshelf 数据库：

MySQL
1
2
CREATE DATABASE bookshelf;
USE bookshelf;

创建一个数据库用户 reader，对 bookshelf 进行访问：

MySQL
1
2
GRANT ALL ON bookshelf.* TO 'reader'@'localhost'
  INDETIFIED BY 'passwd';

在当前使用的数据库中创建表 books，并检查该表：

MySQL
1
2
3
4
5
6
7
CREATE TABLE books (
  isbn CHAR(13),
  author VARCHAR(128),
  title VARCHAR(128),
  category VARCHAR(32),
  year SMALLINT) ENGINE MyISAM;
DESCRIBE books;

删除表，检查当前库中的所有表：

MySQL
1
2
DROP TABLE books;
SHOW tables;

重命名表为 works 作品、添加新列 pages 页码、删除页码列：

MySQL
1
2
3
ALTER TABLE books RENAME works;
ALTER TABLE books ADD pages SMALLINT UNSIGNED;
ALTER TABLE books DROP pages;

 

数据

数据类型支持 CHAR、VARCHAR、BINARY、TEXT、BLOB、数值型、DATE 和 TIME、AUTO_INCREMENT 自增。

为 books 表添加自增列 id，注意如果表中已经设置了 isbn 为主键，则无法添加该列，自增列必为主键：

MySQL
1
2
ALTER TABLE books
  ADD id INT UNSIGNED NOT NULL AUTO_INCREMENT KEY;

 

增加数据 INSERT

加入一本新书：

MySQL
1
2
INSERT INTO bookshelf.books (isbn, author, title, category, year)
  VALUES ('9787121022982', '史蒂夫·迈克康奈尔 ', '代码大全（第2版）', '编程', '2006');

 

删除数据 DELETE

删除某本书：

MySQL
1
DELETE FROM books WHERE isbn = '9787121022982';

 

（改）更新数据 UPDATE

更改某书的部分信息，WHRER 返回条件成立的结果用于缩小范围：

MySQL
1
2
UPDATE books SET author = 'Steve McConnell', category = 'code'
  WHERE isbn = '9787121022982';

 

查询数据 SELECT

查询并显示全部图书信息：

MySQL
1
SELECT * FROM books;

查询图书数量：

MySQL
1
SELECT COUNT(*) FROM books;

对作者进行查询显示，重复数据只显示一次：

MySQL
1
SELECT DISTINCT author FROM books;

查询书名中包含指定文本 “and”，并显示作者和书名信息：

MySQL
1
2
SELECT author,title FROM books
  WHERE title LIKE "%and%";

限制返回结果的行数：

MySQL
1
SELECT title FROM books LIMIT 3;

对结果逆排序：

MySQL
1
SELECT author,title FROM books ORDER BY title DESC;

关联两张表，使用 SELECT … FROM … WHERE，或 JOIN … ON，或 NATURAL JOIN 结构

 

索引和主键

索引能够提高数据的搜索速度，需要为待索引列建立索引（会占用额外空间），如下限制为 author 列的前20个字符进行索引：

MySQL
1
ALTER TABLE books ADD INDEX(author(20));

主键给每行数据唯一的标示符，在 books 表中选用 isbn 为主键，声明 isbn 列为主键必须保证数据唯一：

MySQL
1
ALTER TABLE books ADD PRIMARY KEY(isbn);

通常在创建表时，直接添加索引和主键

如果对某字段建立了全文索引，则可以通过 MATCH … AGAINST 来高级查询

 

数据库设计

  • 规范化：第二范式 or 第三范式
  • 联系：一对一、一对多、多对多
