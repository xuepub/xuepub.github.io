使用 IntelliJ IDEA 开发 Spring Web

on 2016/06/02
  • 应用
by szzz

 准备

  • JDK 8
    • 系统配置 – JAVA_HOME、Path、CLASSPATH
  • Tomcat 8.0
  • IntelliJ IDEA Ultimate 2016.1
    • Maven 内置
  • Gradle
    • 系统配置- GRADLE_HOME

 

IntelliJ 常规配置

  • 代码字体和配色 File – Setting – Editor – Colors & Fonts
  • 服务器设置 File – Setting – Build, Excution, Deployment – Application Servers
    • 添加 Tomcat
  • File – Setting – Version Control – Github

 

创建工程

  • Create New Project – Spring Initializr
    • Maven war 1.8 Java
    • Group = package: com.xuepub
    • Name = Artifact
    • 依赖 (Dependencies): Web Thymeleaf
  • 创建文件后
    • pom.xml – Add as Maven Project
    • Maven 开始工作

 

配置工程

  • File – Project Structure – Artifacts
    • 修改 [name]:war
    • 修改 [name]:war exploded
  • Run – Edit Configurations
    • 添加 Tomcat Server – Local
      • 命名为 tomcat
      • Server – Before launch: Make, Activate tool window 加入 Build Artifacts 2 项
      • Deployment – 添加 [name]:war  或  [name]:war exploded 2 选 1

 

运行

  • Run – Run… – Tomcat – 能成功运行
    • 此时启动本机 Java Web 容器 Tomcat（采用 War 发布）
    • War 包可部署至产品服务器
    • 并非启动嵌入式 Tomcat（以 Jar 发布）
  • 添加控制器和视图代码 https://spring.io/guides/gs/serving-web-content/
    • 控制器位置 src/main/java/com.xuepub/GreetingController.java
    • 视图位置 src/main/resources/templates/greeting.html
  • 重新启动 Tomcat
  • 访问 localhost:8080/greeting?name=abc，得到结果 Hello, abc!
